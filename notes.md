# Manual

Create the Python virtual environment

```bash
pytnon -m venv .venv
./.venv/Scripts/Activate.ps1
python.exe -m pip install --upgrade pip
pip install mkdocs-material mkdocs-glightbox
```

```bash
mkdocs serve
```
